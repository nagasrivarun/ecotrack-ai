# EcoTrack AI – Carbon Footprint Awareness Platform

A modern React + Vite application that helps users calculate, track, and reduce carbon emissions with analytics, recommendations, and achievements.

## Features

- Carbon calculator for transport, electricity, food, and flights
- Interactive dashboard with pie and bar charts
- Personalized recommendations and achievements
- Local storage persistence
- Dark theme with glassmorphism design

## Setup

1. Install dependencies:
   `npm install`
2. Run locally:
   `npm run dev`
3. Build for production:
   `npm run build`

## Deploy

Deploy to Vercel by connecting the repository and using the default build command `npm run build`.
